Add Course Screen Fields  
- Title

- Price

- Type  
   * Drop-down contains:

     * Advanced   
- Description  
   * Any characters  
   
   * Up to 250 characters   
- Submit button  
   * Orange color  
**Add Course Flow**  
- Users can add courses from the Home screen.
