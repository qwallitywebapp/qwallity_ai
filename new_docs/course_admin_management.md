# Admin: Edit and Delete Courses  
- **Access:** Admin clicks the Course Title (Link) in the course table.  
- **Edit Screen:**  
  - Fields: Title, Description, Submit button.  
  - Success: Shows "Your changes are done" and redirects to Courses page.  
- **Delete Action:**  
  - Admin clicks Delete button.  
  - Result: Redirect to Courses page with "Course is deleted" message.  
- **Constraint:** Admin users cannot see or use the "Buy" button.
