# Admin: Adding New Courses  
- **Access:** Green "Add Course" button on Home screen.  
- **Fields:**  
  - Title: Alphabetical characters only, up to 50 characters.  
  - Price: Numbers only (int/float/decimal).  
  - Type: Dropdown (Fundamental or Advanced).  
  - Description: Any characters, up to 250 characters.  
- **Action:** Orange Submit button. Redirects to Home page after success.
