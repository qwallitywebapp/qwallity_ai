# Receipt Download (Print Receipt)  
- **Permissions:** Only available for non-admin users.  
- **Behavior:**   
  - Clicking before checkout: Page reloads, no file downloaded.  
  - Clicking after checkout: Downloads `receipt.docx` with last checkout data.  
- **Required Fields in Receipt:**  
  - Username, Amount, Date/Time of refill, Payment Method, and Account Balance after refill.
