# Course Catalog and Weather Widget  
- **Navigation:** Users view courses separated into "Advanced" and "Fundamental" sections.  
- **Section View:** Displays table with Author, Title, and Buy columns.  
- **Weather Icon:**   
  - Located in the top right of the navigation bar.  
  - Hovering displays the temperature in Celsius (C) based on location.  
  - Refreshes when the page refreshes.
