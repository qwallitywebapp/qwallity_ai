# Password Recovery (Forget Password)  
- **Flow:**   
  1. Click 'Forget Password' -> Change Password screen.  
  2. Enter valid Email -> Display 'Secure code sent to your email'.  
  3. Click 'Change Password' link in email -> Navigate to screen with secure code.  
- **Change Password Fields:**  
  - Code field: Minimum 6 characters.  
  - Password field: 6-14 characters.  
  - Submit button.
