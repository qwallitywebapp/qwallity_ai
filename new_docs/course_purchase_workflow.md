# Course Purchasing Logic  
- **Access:** Only for non-admin users via the "Buy" button.  
- **Buy Screen Fields:**  
  - Course title-price.  
  - Course description.  
  - Orange "Buy" button.  
- **Post-Purchase Logic:**  
  - Insufficient Balance: Redirect to "My Courses" screen.  
  - Sufficient Balance: Course is added to "My Courses" screen for the user.
