Course Viewing  
- Both admin and non-admin users can view all courses.  
- Users can see separated courses on the Course page:  
    * Advanced  
    * Fundamental  
 **Course Section Navigation**  
   
    * Selecting Advanced or Fundamental redirects the user to the corresponding course section.  
- The following columns are visible:  
   
    * Author  
    * Title  
    * Buy  
 **Buy Course Screen**  
   
    After clicking on Buy, the user is redirected to a separate screen containing:  
- Course title-price  
- Course description  
- Buy button (Orange color )
