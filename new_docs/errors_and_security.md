**Invalid Login**  
 If username or password is incorrect:  
- Message displayed:  
   * "Username or password is incorrect"   
**Account Security**  
- System blocks the account after 6 failed login attempts.   
**Toggle Behavior**  
- Password visibility toggle allows viewing the entered password.   
**Cancel Button Behavior**  
- Resets both fields:  
   
   * Username  
   
   * Password
