# Login Interface and Controls  
- **Fields:** Username, Password.  
- **Features:**   
  - Toggle password visibility (eye icon).  
  - Login Button.  
  - Cancel Button: Resets all entered data in both fields.  
  - Forget Password Link: Redirects to password recovery flow.  
- **Post-Login UI:** Redirects to Home page.  
- **Success Message:** "Welcome {registered name}".
